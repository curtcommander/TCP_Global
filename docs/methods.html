(async) makeFolder(params) → {string}

Source:
  • lib/upload.js, line 81
Makes a new folder in Google Drive. If both params.parentId and params.parentName aren't specified, the folder will be created in the root folder.
Example
// make a new folder named "newFolder" in the folder "parentFolder"
utilsGDrive.makeFolder({
  folderName: "newFolder",
  parentName: "parentFolder"
});

// make a new folder named "newFolder" in the root folder
utilsGDrive.makeFolder("newFolder");
Parameters:
Name Type Description
params Object | string Function parameters. A string containing the name of the folder to be created may be passed instead. In this case, the folder will be created at the root folder
Properties
Name Type Attributes Description
folderName string Name of folder to be created
parentIdentifiers Object <optional>
 Same as identifiers in other methods (e.g. getMime) but for the parent folder where the new folder will be created

 Whether to overwrite an existing folder. Default value is false

Example

  localPath: "path/to/folderName",

});

utilsGDrive.upload("path/to/folderName");
