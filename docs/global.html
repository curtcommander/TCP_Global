(async) listChildren(identifiers, fieldsopt) → {Array}

Source:
  • lib/utils.js, line 278
Gets data on files in a folder.
Example
// check if folder has any subfolders
utilsGDrive.listChildren({folderName: "Folder Name"}, 'files(mimeType)')
.then(data => {
 for (const d of data) {
   if (d.mimeType === "application/vnd.google-apps.folder") {
     console.log("Folder has subfolders");
     break;
   }
 }
});
Parameters:
Name Type Attributes Default Description
identifiers Object An object identifying the folder with either folderId or folderName. parentId and parentName properties may also be included
fields String <optional>
 files(name, id, mimeType) The data fields to request. Format should follow the pattern 'files(field1, field2,...)'
Returns:
Each entry in the array returned is an object containing data on a file that resides in the folder specified
Type
Array
