(async) listChildren(folderParams, parentParamsopt, fieldsopt) → {Array}

Source:
  • lib/utils.js, line 301
Gets data on files in a folder.
Example
// check if folder has any subfolders
utilsGDrive.listChildren({folderName: "Folder Name"}, null, 'files(mimeType)')
.then(data => {
 for (const d of data) {
   if (d.mimeType === "application/vnd.google-apps.folder") {
     console.log("Folder has subfolders")
     break;
   }
 }
});
Parameters:
Name Type Attributes Default Description
folderParams Object An object identifying the folder with either folderId or folderName
parentParams Object <optional>
 An object identifying the parent with either parentId or parentName
fields String <optional>
 files(name, id, mimeType) The data fields to request. Format should follow the pattern 'files(field1, field2,...)'
Returns:
Each entry in the array returned is an object containing data on a file that resides in the folder specified
Type
Array
